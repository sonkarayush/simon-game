# 🎮 Simon Game

A web-based recreation of the classic memory test game "Simon", built entirely with HTML, CSS, and Vanilla JavaScript. Test your memory by repeating an ever-growing sequence of colors!

## ✨ Features
* **Dynamic Level Progression:** The sequence gets longer and more challenging with every successful round.
* **Visual Feedback:** Distinct flash animations for both the game's sequence and user interactions.
* **Strict Validation:** Real-time array comparison to ensure the user input matches the game sequence exactly.
* **Score Tracking:** Displays your final level reached upon Game Over.
* **Modern UI:** Clean, responsive layout utilizing CSS Flexbox.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure and element targeting.
* **CSS3:** Flexbox layout, custom borders, and CSS animations/transitions for the flash effects.
* **JavaScript (ES6):** DOM manipulation, event listeners, array management, and asynchronous timing functions (`setTimeout`).

## 🚀 How to Play
1. Press **any key** on your keyboard to start the game.
2. Watch the boxes carefully. One of them will flash.
3. Click the exact box that flashed.
4. If correct, you will advance to Level 2. The game will replay the first flash, and add a second one.
5. Repeat the sequence from the beginning every time. 
6. One wrong click, and it's **Game Over!** ## 💻 Run Locally
